# Banking System Simulator - JAVA

A Java-based banking program developed for an Object-Oriented Programming (OOP) semester project. 

## Features
- Banking simulation logic based on Object-Oriented principles.
- Code is contained within the `src/Banking_Project_Spandan` package.

## Getting Started

1. Open this project in your preferred Java IDE (like IntelliJ IDEA, which has the associated `.idea` folder and `.iml` file here).
2. Navigate to the `src` folder to locate the main source code.
3. Build and run the project to test the banking simulator.
